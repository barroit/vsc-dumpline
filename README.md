dumpline
========

dumpline is the fastest code snipping tool in the vscode extension.

How to use
==========

Select the region you want to dump:

![step 1](https://raw.githubusercontent.com/barroit/dumpline/refs/heads/master/image/barroit/step_1.webp)

Execute `dumpline: copy and preview result`:

![step 2](https://raw.githubusercontent.com/barroit/dumpline/refs/heads/master/image/barroit/step_2.webp)

You get the result:

![step 3](https://raw.githubusercontent.com/barroit/dumpline/refs/heads/master/image/barroit/step_3.webp)

There are three buttons at the top right. First one is for copying, second one
opens the directory where image lives, last one opens image itself.
